## Bones and Skeletons

##### Basically a skeleton (BABYLON.Skeleton) contains a heirarchy of bones (BABYLON.Bone). 
##### There are constructors like:
##### - BABYLON.Skeleton = function (name, id, scene)
##### - BABYLON.Bone = function (name, skeleton. parentBone, matrix)
